Once an attacker is inside a system, they may perform various actions to exploit the compromised system further.

Stage 5

Man in the Midle Attack

ARP stands for Address Resolution Protocol. It is a communication protocol used in computer networks to map an IP address (Internet Protocol address) to a physical machine address, known as a Media Access Control (MAC) address. The MAC address is associated with the network interface card (NIC) of a device


When a device on a local network wants to communicate with another device, it needs to know the MAC address of the target device. ARP helps in this process by resolving the IP address to the corresponding MAC address. If a device knows the MAC address, it can then send data frames directly to the intended recipient on the local network


ARP operates at the data link layer (Layer 2) of the OSI model. It is essential for the proper functioning of local area networks (LANs) and is a fundamental part of the TCP/IP protocol suite.

Bettercap tool

BetterCAP is a powerful, open-source network attack and penetration testing framework that is designed for various network-related tasks, particularly those related to Man-In-The-Middle (MITM) attacks. It's written in the Go programming language and provides a modular architecture that allows users to extend its functionality through plugins

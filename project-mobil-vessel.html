Project Summary

I led a multidisciplinary technical team to deliver a full vessel IT upgrade. The work included identity management (Entra ID) for all crew, provisioning of virtualization (VMware ESXi) hosting Windows Server 2016 instances, setting up iLO for hardware management, configuring enterprise backups using Veeam, and deploying an end-to-end CCTV solution using Hikvision NVRs (32-/64-channel) and iVMS desktop/cloud integration to achieve 90 days of video retention. We also deployed redundant internet connectivity (Intellian / Sailor / Starlink) and integrated Teams telephony for reliable on/offshore communications.

My responsibilities

  • Project lead for planning, team management and execution (5-person team)
  • VMware ESXi host provisioning and Windows Server deployment
  • Hikvision iVMS and NVR architecture to enable 90-day retention and cloud sync
  • iLO setup for remote hardware management and Veeam backup orchestration
  • Network config: SADP for camera addressing, Cisco switch VLANs, and failover links
  • Onsite coordination with shipyard welders for antenna mounts and cable routing

Technologies used

Hikvision iVMS VMware ESXi Windows Server 2016 Veeam Cisco Switches iLO / Intellian / Sailor Entra ID

Impact & Outcome

Before: 30-day CCTV retention, limited remote access and ad-hoc backups.

After: 90-day retention via NVR + cloud sync, centralized identity & management for onboard staff, reliable remote hardware management and scheduled backups with Veeam—improving compliance and operational visibility.

Business value: Standardized IT onboard systems, reduced risk, and faster incident response—delivered in 2 weeks with minimal operational disruption.
